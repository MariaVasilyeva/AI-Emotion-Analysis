# Описание датасета Student Performance & Academic Trends Dataset

**Название файла:** `ai_impact_student_performance_dataset.csv`

**Краткое описание:**
Данный датасет предназначен для анализа того, как инструменты искусственного интеллекта (ИИ) влияют на учебный процесс, поведение и академические результаты учащихся. Набор данных содержит 8 000 записей и 26 признаков, сочетающих демографические данные, паттерны использования ИИ, данные об учебной активности и метрики успеваемости.

## Основные характеристики

**Объём:** 8 000 строк, 26 столбцов.

**Источник данных:** Kaggle — *Student Performance & Academic Trends Dataset*.
Ссылка: [https://www.kaggle.com/datasets/ankushnarwade/ai-impact-on-student-performance](https://www.kaggle.com/datasets/ankushnarwade/ai-impact-on-student-performance)

**Лицензия:** Attribution 4.0 International (CC BY 4.0)

**Целевые переменные:** `final_score` (непрерывная), `passed` (бинарная), `performance_category` (категориальная).

## Структура данных

Ниже приведён полный перечень столбцов с описанием и типом данных.

### 1. Демографическая информация
| Столбец | Тип данных | Описание |
| :--- | :--- | :--- |
| `student_id` | int64 | Уникальный идентификатор студента |
| `age` | int64 | Возраст студента (в годах) |
| `gender` | object | Пол (Male, Female, Other) |
| `grade_level` | object | Текущий уровень обучения / класс |

### 2. Паттерны использования ИИ (Поведенческие факторы)

| Столбец | Тип данных | Описание |
| :--- | :--- | :--- |
| `uses_ai` | int64 | Бинарный флаг использования ИИ в обучении (1 = Да, 0 = Нет) |
| `ai_usage_time_minutes` | int64 | Общее время работы с ИИ-инструментами в минутах |
| `ai_tools_used` | object | Названия используемых инструментов (например: ChatGPT, Copilot, Claude) |
| `ai_usage_purpose` | object | Основная цель использования (подготовка к экзаменам, конспектирование, кодинг и т.д.) |
| `ai_dependency_score` | int64 | Оценка зависимости студента от AI-инструментов (метрика психологической привязанности) |
| `ai_generated_content_percentage` | int64 | Процент выполненных заданий или работ с использованием искусственного интеллекта |
| `ai_prompts_per_week` | int64 | Среднее количество запросов к ИИ в неделю |
| `ai_ethics_score` | int64 | Показатель соблюдения этических норм при использовании ИИ |

### 3. Академические показатели и эффективность

| Столбец | Тип данных | Описание |
| :--- | :--- | :--- |
| `last_exam_score` | int64 | Балл за последний пройденный экзамен |
| `assignment_scores_avg` | float64 | Средний балл за домашние и проверочные работы |
| `attendance_percentage` | float64 | Процент посещаемости занятий |
| `concept_understanding_score` | int64 | Оценка понимания ключевых концепций курса |
| `improvement_rate` | float64 | Темп улучшения академических результатов в течение семестра |
| `class_participation_score` | int64 | Оценка активности и вовлеченности на занятиях |
| `final_score` | float64 | **Итоговый балл** (агрегированная метрика, включающая результаты экзаменов, учебных заданий и участия в образовательном процессе) |
| `passed` | int64 | **Статус прохождения курса** (1 = Сдал, 0 = Не сдал) |
| `performance_category` | object | **Категория успеваемости** (Low, Medium, High) |

### 4. Образ жизни и привычки (Контекстуальные данные)

| Столбец | Тип данных | Описание |
| :--- | :--- | :--- |
| `study_hours_per_day` | float64 | Среднее количество часов обучения в день |
| `study_consistency_index` | float64 | Индекс стабильности учебных привычек (высокое значение = системный подход) |
| `sleep_hours` | float64 | Средняя продолжительность сна в сутки |
| `social_media_hours` | float64 | Время, затрачиваемое на социальные сети ежедневно |
| `tutoring_hours` | float64 | Время дополнительных занятий с репетитором |
